Yummie Catcher
Ловец вкусняшек

Описание игры 
Вкусняшки падают прямо с неба. Ловец должен поймать 100 вкусняшек каждого вида(пицца, кола, торт, чипсы) как можно быстрее.

Управление
→ - перемешение героя вправо
← - перемешение героя влево
1-2 - смена персонажа
q - выход

Для запуска игры используется файл yummie_catcher.py.
